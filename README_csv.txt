Instrucciones CSV (móvil):
- Coloca tu archivo CSV en: data/turnos.csv
- Cabeceras admitidas (con alias): hotel, semana_lunes, empleado, fecha, turno
- Separador coma o punto y coma. Codificación UTF-8.
- Si hay varios CSV, opcionalmente añade data/turnos_extra.csv o data/turnos2.csv.
- Si existe window.FULL_DATA (data.js), se usará primero. El CSV es fallback automático.
